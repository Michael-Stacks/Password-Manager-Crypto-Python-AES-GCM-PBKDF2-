# Password-Manager-Crypto-Python-AES-GCM-PBKDF2-
A simple and secure command-line Password Manager built in Python.   It uses SHA-256 for master password hashing, PBKDF2-HMAC-SHA256 for key derivation,   and AES-256-GCM to encrypt stored passwords.   Includes bonus features such as strong password generation and entry deletion.
